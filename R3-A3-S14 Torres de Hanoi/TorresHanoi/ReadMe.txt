**Instrucciones**

Se debe realizar una investigación acerca de Las torres de Hanoi.
Implementar en el lenguaje de su elección el ejercicio de las Torres de Hanoi, donde se evidencie el funcionamiento del algoritmo.
Se debe evidenciar en la aplicación el uso de funciones recursivas
El ejercicio se debe realizar en el grupo de trabajo conformado.

**Condiciones de entrega**

El ejercicio debe estar codificado en lenguaje de programación de su elección
Se debe generar el archivo fuente en un código de alto nivel
Se debe entregar el proyecto generado desde el IDE de su elección
Se debe realizar la documentación interna del código mediante comentarios en cada una de las partes del código
