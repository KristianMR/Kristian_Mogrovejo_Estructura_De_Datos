Se debe realizar una investigación acerca del funcionamiento de las estructuras dinámicas de tipo LIFO o Pila (último en entrar, primero en salir) y de tipo FIFO o Cola (Primero en entrar, primero en salir)
Crear en el lenguaje de su elección un programa implemente el funcionamiento de una Bicola, el cual tendrá el siguiente menú de opciones:
- Insertar por la derecha
- Insertar por la izquierda
- Atender por la derecha
- Atender por la izquierda
- Listar.
- Salir.
Para la creación de este programa se deberá usar como estructura de datos una lista simple

**Condiciones de entrega**

El ejercicio debe estar codificado en lenguaje de programación de su elección
Se debe generar el archivo fuente en un código de alto nivel
Se debe entregar el proyecto generado desde el IDE de su elección
Se debe realizar la documentación interna del código mediante comentarios en cada una de las partes del código
